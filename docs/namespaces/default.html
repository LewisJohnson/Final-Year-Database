get_config_json()

get_config_json(string  $key = null, string  $value = null) : string|boolean|null

This function is used to get or set the department specific JSON configuration file or a value in the said file.

Parameters

string $key

The key to search for.
string $value

The value to set.

Returns

string|boolean|null —

If $key is null, the whole file is returned. If $value is null, the key-value is returned. If $key and $value is set, a boolean is returned.
